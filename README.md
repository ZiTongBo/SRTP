使用Unity3D引擎及C#脚本开发的VR室内装修设计
凝视可编辑物体，物体会闪烁 
凝视3秒后选中物体，可执行移动，旋转，删除操作
不可编辑物体包括（门，墙，房顶，灯）
凝视门3秒后进入下一房间
m键打开主菜单，n键添加新物体（若按下无反应请切换输入法）
导入户型功能暂未实现
目前的3种户型实为同一种，共两房间
保存设计功能将截图保存在项目根目录的Screenshot文件夹中
目前暂未实现物体碰撞检测
